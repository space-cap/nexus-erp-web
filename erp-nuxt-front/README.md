# ERP Nuxt Mock Frontend

Delphi 기반 ERP의 웹 전환을 검토하기 위한 Nuxt mock 프론트엔드입니다. 백엔드는 아직 연결하지 않았고, `app/data/mock/erp.ts`와 `useMockErpStore.ts`의 mock 데이터를 사용합니다.

## 실행

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 3001
```

접속 주소:

```text
http://127.0.0.1:3001/
```

## 주요 화면

- 대시보드
- 거래처 목록, 신규 등록, 상세, 수정
- 품목 목록, 신규 등록, 상세, 수정
- 재고 현황, 재고 입출고
- 수주 목록, 신규 등록, 상세, 수정
- 발주 목록, 신규 등록, 상세, 수정
- 생산 계획 목록, 신규 등록, 상세, 수정
- 처리 이력과 감사 로그 조회
- 사용자 관리
- mock 로그인/권한 선택

## Mock 기능

- `useMockErpStore.ts`에서 화면 간 공유 mock 데이터 관리
- 브라우저 `localStorage` 저장으로 새로고침 후 mock 데이터 유지
- 수주 등록/수정 시 예약 재고 반영
- 수주 완료 시 출고 이력 생성
- 재고 부족 품목에서 발주 등록 폼 자동 채움
- 발주 완료 시 재고 입고 반영
- 수주 상세에서 생산 지시 생성
- 생산 진행 시 자재 출고 반영
- 생산 완료 시 제품 입고 반영
- 수주/발주/생산/재고 처리 이력 자동 기록
- `useMockAuth.ts`에서 역할별 메뉴 및 버튼 노출 제어

## 빌드
